## How to run this project
Clone this repo
```bash
git clone https://github.com/Alex-dll/login-app
```
Go to project directory
```bash
cd login-app
```
Install the dependencies
```bash
npm install | npm i
```
Start the aplication
```bash
npm run start
```
See in your device using Expo Go
